HR Resume Registry Assistant
Версия: 1.0.0
Разработчик: Nikita Karpuk

Быстрый старт
1. Запустите HR Resume Registry Assistant из меню Пуск или ярлыка на рабочем столе.
2. Дождитесь автоматического открытия браузера.
3. Если браузер не открылся, скопируйте локальный адрес из окна приложения. Обычно это http://127.0.0.1:8000.
4. Не закрывайте окно приложения до завершения работы.

Рабочий процесс
1. Загрузка реестра: загрузите Excel-файл с листом "реестр".
2. Загрузка резюме: загрузите PDF, DOCX или TXT.
3. Сопоставление: запустите подбор резюме к строкам реестра.
4. Ручная проверка: подтвердите или отклоните спорные совпадения.
5. Качество данных: проверьте предупреждения по реестру.
6. Экспорт: сформируйте обогащенный Excel-файл.

Рабочие данные
Папка данных:
%LOCALAPPDATA%\HRResumeRegistryAssistant\project_files\

В этой папке хранятся:
- база SQLite;
- исходные реестры;
- исходные резюме;
- готовые переименованные резюме;
- экспортированный Excel;
- локальный .env с настройками ИИ.

Безопасность
- Не передавайте API-ключи третьим лицам.
- Для обработки конфиденциальных HR-данных используйте только доверенный компьютер.
- Перед отправкой выгрузок наружу проверяйте персональные данные и состав колонок.
